En el directori alumnesWebs s'han de crear les webs dels alumnes.
Crea una carpeta amb el teu nomCogonm i crea la teva web.
La Web ha de contenir:
-arxiu--> nomCognom.html
Amb un titol amb el teu nom i cognom
Una imatge enllaçada des d'internet
Opciona: Afegit un arxiu style.css amb algun estil per la teva web

IMPORTANT: Has de modificar l'arxiu index.html perquè linki correctament a la teva web

Com fer-ho:
1.- Enllaça amb el repositori remot .......
2.- Fes un pull per descarregar la branca master i la branca gh-pages
3.- Situa't en la branca gh-pages i crea una branca amb el nom "devNomCognom"
4.- Situa't a la branca "DevNomCognom"
5.- Crea la teva web dins la carpeta "alumnesWebs"
6.- Opcional: pots afegir un style.css per donar estil a la web i comprova que funcioan correctament
7.- Fes un commit amb les modificacions
8.- Modifica l'arxiu index.html perquè enllaci correctament la teva web i comprova que enllaça correctament.
9.- Fes un commit amb les modificacions de l'index
10.- Fes un merge de la branca "devNomCognom" a la branca gh-pages
11.- Fes un push per la branca "devNomCognom" i la branca gh-pages
12.- Ves al repositori de git hub ..... i comprova que tot s'ha pujat correctament
13.- Visualitza la teva web a través dle següent enllaç
